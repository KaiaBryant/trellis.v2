Welcome to Trellis! We are a one stop shop for those looking to begin gardening. 

This full stack e-commerce is built using Vite React to render front-end and JS functionality, a database that contains products and collects users contact information, server functions and media responsivness for mobile, tablet and desktop. 

Special features include a working slider on homepage for short cut access to shop by categories, a filter on product page to encourge ease with shopping for products, for validation on the contact page, logo that routes the uder to home page, and lastly a cohesive footer with social media icons and links.
